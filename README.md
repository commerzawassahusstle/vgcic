# VGCIC Teentalk Website

A modern, responsive website for VGCIC Teentalk event.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Sections Included**:
  - Hero section with call-to-action
  - About section
  - Speakers section
  - Event schedule
  - Registration form
  - Contact information
  - Footer with social links

## Getting Started

1. Open `index.html` in a web browser
2. Customize the content with information from this year's flyer
3. Add images and assets as needed

## Customization

### To update content:
- Edit `index.html` to change text content
- Update colors in `styles.css` by modifying CSS variables in `:root`
- Add speaker images and information in the speakers section
- Update schedule with actual event times

### To add images:
1. Create an `images` folder
2. Add your images
3. Update image paths in `index.html`
